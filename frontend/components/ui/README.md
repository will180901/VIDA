# UI Components

Ce dossier contient les composants UI réutilisables du projet VIDA.

## Composants Actifs

Les composants suivants sont actuellement utilisés dans l'application :

- **AppointmentModal** - Modal de prise de rendez-vous
- **ContactModal** - Modal de formulaire de contact
- **GradientMeshBackground** - Fond animé avec particules
- **HeroSlider** - Carrousel de slides hero
- **ImageWithFallback** - Image avec fallback en cas d'erreur
- **PageSkeleton** - Skeleton loader pour le chargement de page
- **ScrollIndicator** - Indicateur de scroll
- **SectionIndicator** - Indicateur de section
- **Toast** - Notifications toast

## Composants Réservés

Les composants suivants sont réservés pour les modules futurs (modules 5-11) et ne doivent pas être supprimés :

- **Badge** - Badges pour affichage de statuts
- **Button** - Boutons réutilisables
- **Card** - Cartes de contenu
- **LottieAnimation** - Animations Lottie (actuellement non utilisé, react-lottie-player retiré)

## Composants d'Authentification

- **ProtectedRoute** - Route protégée pour l'authentification (réservé pour modules futurs)
- **AuthCard** - Carte d'authentification
- **AuthInput** - Input pour formulaires d'authentification

## Notes

- Ne pas supprimer les composants réservés même s'ils ne sont pas utilisés actuellement
- Ces composants seront utilisés lors de l'implémentation des modules 5 à 11
- Pour toute question sur l'utilisation d'un composant, consulter la documentation du module correspondant
