# Guide Administrateur - VIDA

## Accéder à l'Admin

1. Se connecter avec un compte admin
2. Accéder à /admin/dashboard

## Dashboard

Le tableau de bord affiche :
- Nombre de rendez-vous aujourd'hui
- Rendez-vous en attente de confirmation
- Statistiques de la semaine

## Gestion des Rendez-vous

### Confirmer un Rendez-vous

1. Aller dans "Rendez-vous"
2. Cliquer sur un rendez-vous en attente
3. Cliquer sur "Confirmer"

### Refuser un Rendez-vous

1. Cliquer sur le rendez-vous
2. Cliquer sur "Refuser"
3. Entrer le motif du refus
4. Le patient sera notifié par email

### Proposer une Nouvelle Date

1. Cliquer sur le rendez-vous
2. Cliquer sur "Proposer une nouvelle date"
3. Sélectionner une nouvelle date et heure
4. Envoyer la proposition

## Gestion des Patients

### Lister les Patients

1. Aller dans "Patients"
2. Voir la liste complète
3. Filtrer par nom, email, statut

### Consulter un Dossier Patient

1. Cliquer sur le patient
2. Voir :
   - Informations personnelles
   - Historique des rendez-vous
   - Notes médicales

## Notifications

Le système notifie automatiquement :
- Confirmation de rendez-vous
- Rappel 24h avant
- Annulation
- Proposition de nouvelle date

## Rapports

### Statistiques

1. Aller dans "Rapports"
2. Choisir la période
3. Voir :
   - Nombre de consultations
   - Taux de présence
   - Revenus

### Export

1. Cliquer sur "Exporter"
2. Choisir le format (CSV, PDF)
3. Télécharger le rapport
