# Guide d'Utilisation - VIDA

## Présentation

Ce guide décrit comment utiliser la plateforme VIDA selon votre rôle.

## Rôles Utilisateurs

| Rôle | Description | Accès |
|------|-------------|-------|
| **Patient** | Utilisateur du service | Espace patient, prise de RDV |
| **Staff** | Personnel administratif | Gestion RDV, patients |
| **Médecin** | Personnel médical | Dossiers médicaux |
| **Admin** | Administrateur | Accès total |
