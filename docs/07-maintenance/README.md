## 🔄 Tâches de maintenance

### Quotidiennes (automatiques)

- ✅ Backup de la base de données (2h du matin)
- ✅ Nettoyage des tokens expirés
- ✅ Envoi des rappels de rendez-vous

### Hebdomadaires

- 🔍 Vérifier les logs d'erreur
- 📊 Analyser les métriques de performance
- 🔐 Vérifier les tentatives de connexion suspectes

### Mensuelles

- 🗄️ Vérifier les backups (test de restauration)
- 📦 Mettre à jour les dépendances
- 🧹 Nettoyer les logs d'audit > 90 jours
